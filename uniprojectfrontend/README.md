# University App — Frontend

*React frontend for the University Management REST API*

---

Built with React, React Router, Axios, Tailwind CSS, and Framer Motion. Consumes the Django REST Framework backend at `http://127.0.0.1:8000`.

## Running Locally

Ensure the backend server is running first, then:

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Available Scripts

| Command | Description |
|---|---|
| `npm start` | Start development server |
| `npm test` | Run test suite |
| `npm run build` | Build for production |
